# 00. プロジェクト概要

## 目的
- LifeManager を通じて、個人のタスク・習慣・予定を一元管理する

## 機能

- 一週間の運動履歴や行動を健康の観点から評価してくれる機能
    - 運動量・睡眠
- 食べ物のメニューを行動履歴から提案してくれる
- テックブログの記事(Qiitaから最新情報を集める)

## 使用する外部機能
- HealthAutoExports (iphpneのヘルスケア情報から自身の行動情報をデータとして転送してくれる)
- Messaging API(Lineに通知が来るようにする)
- OpenAI API (行動履歴を評価・自炊メニューを考案してくれる)
- Qiita API (Qiitaから記事のリンクを取得してくれる)
- X API (Xから記事のリンクを取得してくれる)
